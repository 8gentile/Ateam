# API Endpoints

## HTML API
###Root
  - GET / -loads the React web app

## JSON API
###Users
  - POST /api/users
  - PATCH /api/users
###Session
  - POST /api/session
  - DELETE /api/session
###Teams
  - GET api/teams
  - POST api/teams
  - GET api/teams/:id
  - PATCH api/teams/:id
  - DELETE api/teams/:id
###Todo Lists
  - GET api/todolists
  - POST api/todolists
  - GET api/todolists/:id
  - PATCH api/todolists/:id
  - DELETE api/todolists/:id
###Todo Items

  - GET api/todoitem/:id
  - PATCH api/todoitem/:id
  - DELETE api/todoitem/:id
###Schedules

###Events
